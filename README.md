## Interactive Console for .NET Core command-line

[![Build Status](https://travis-ci.org/wk-j/dot-console.svg?branch=master)](https://travis-ci.org/wk-j/dot-console)

![](Images/DotConsole.png)

## Installation

```bash
npm install -g dot-console
```

## Usage

Start console 

```bash
dot-console
```

Exit `CTRL + c`
